# 💯 超星学习通满分助手 AI版

## 📖 简介

本脚本是一款基于AI技术的超星学习通自动答题助手，支持任务点自动跳转、章节测验、作业、考试全网检索答案。简答题支持本地AI回答，音频、视频全自动静音播放，具有可视化参数配置功能。
本脚本为个人修改版，非原作者发布。原脚本地址：[Tampermonkey](https://greasyfork.org/scripts/436994)
**版本**: 2.2.0
**适用平台**: 超星学习通 (chaoxing.com)

## 🤖 AI配置指南

### 本地AI配置（LM Studio）

1. **下载安装 LM Studio**
   
   - 访问 [lmstudio.ai](https://lmstudio.ai/)
   - 下载并安装

2. **启动API服务**
   
   - 打开 LM Studio
   - 选择或下载一个模型（推荐：Qwen、Llama等中文友好的模型）
   - 启用本地服务器
   - 确保端口为 1234

3. **脚本配置**
   
   ```
   AI API地址: http://127.0.0.1:1234/v1/chat/completions
   API Key: lm-studio
   模型名称: 根据LM Studio显示的模型名称填写
   ```

### 在线AI配置（智谱AI为例）

1. **获取API Key**
   
   - 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
   - 注册账号并获取API Key

2. **脚本配置**
   
   ```
   AI API地址: https://open.bigmodel.cn/api/paas/v4/chat/completions
   API Key: 您的API Key
   模型名称: glm-4-flash 或其他可用模型
   编辑脚本在头部添加`// @connect      open.bigmodel.cn`
   否则脚本无法访问对应API
   ```
# ！！！编辑脚本在头部添加`// @connect      open.bigmodel.cn`

### 考试模式

1. 进入考试页面
2. 脚本自动识别当前题目
3. AI给出答案参考
4. 开启"考试自动切换"可自动下一题

## 🎯 答题原理

### 单选题/多选题

1. AI分析题目和选项
2. 返回正确选项的字母（A、B、C、D等）
3. 脚本自动转换并选择对应选项

### 判断题

1. AI判断陈述对错
2. 返回"正确"或"错误"
3. 脚本自动选择对应选项

### 填空题

1. AI识别所有填空位置
2. 按顺序给出答案
3. 脚本自动填充到对应位置

### 简答题/论述题

1. AI分析题目要求
2. 生成简明扼要的答案
3. 显示供参考

### 常见问题

**Q: AI返回的答案不准确怎么办？**
A: 可以尝试：

- 调整AI模型参数（temperature、top_p）
- 更换更强大的模型
- 手动检查并修正答案

**Q: 脚本无法运行怎么办？**
A: 请检查：

- 浏览器脚本管理器是否正常工作
- AI服务是否正常运行
- 网络连接是否稳定
- 脚本版本是否最新

**Q: 如何提高AI回答准确率？**
A: 建议：

- 使用参数量更大的模型
- 确保prompt提示词清晰
- 选择专业领域模型
- 检查API配置是否正确

**Q: 视频无法自动播放？**
A: 可能原因：

- 浏览器阻止了自动播放
- 需要先点击页面任意位置
- 检查浏览器自动播放设置
